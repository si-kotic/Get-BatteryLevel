# Get-BatteryLevel
Get-BatteryLevel displays a graphical representation of the current battery charge level as well as the percentage of charge remaining
## Usage
### Syntax
```powershell
Get-BatteryLevel
```
### Example
On Battery
```
C:\>Get-BatteryLevel
☨▓▓▓▓▓▓▓▓▓▓░ (98%)
```
On A/C Power
```
C:\>Get-BatteryLevel
⚡▓▓▓▓▓▓▓▓▓▓░ (98%)
```
